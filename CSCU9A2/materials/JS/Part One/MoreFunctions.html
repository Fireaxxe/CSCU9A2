Examples using functions with buttons and forms

Using JavaScript forms can call functions. In the form declaration one can select a function to call. When the button below is pressed, the function newText() is called and the text in the text field is updated.

Using a variable

Variables in JavaScript can be local (declared inside a function) or global (declared within a script element but outside a function). Global variables can be referred to anywhere else in the document. This example uses a global variable.

The variable is used to keep a count of how often the button below has been pushed. When the button is pushed, a function is called which increments the count, then sets the text in the form to the new count.
